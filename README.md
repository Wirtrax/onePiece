# 🏴‍☠️ One Piece Explorer

## 📌 About
One Piece Explorer is a web application dedicated to the One Piece universe.  
The project is built primarily with PHP and allows users to explore characters and content with dynamic search powered by AJAX, providing instant results while typing.

## ✨ Features
- 🔍 Live search with AJAX (real-time results while typing)
- 🐘 Backend powered by PHP
- 📚 Database integration
- 📱 Fully responsive (mobile + tablet + desktop)
- ⚡ Fast and lightweight (vanilla JavaScript)
- 🎨 Clean UI focused on content
